Mazel

SEPTEMBRE 2025 - NOVEMBRE 2025

Jeu de labyrinthe interactif

Voir le code

Description du projet:

Mazel est un jeu de labyrinthe à un joueur qui possède deux modes.

- Le premier est un mode de progression accessible lorsqu'un utilisateur de "connecte" via un mot de passe reconue par le système, et accède ainsi à 8 niveaux différents, à difficultées progressives, qu'il doit compéter pour avancer dans les méandres du jardin de Mazel.


- Le second est un mode créatif de labyrinthes, personalisables par le joueur à l'aide de différentes options, comme entre autres : la taille du labyrinthe, la vision dans le labyrinthe, ou encore l'ajout d'd'un ennemi qui pourchasse le joueur jurqu'à la sortie.


Le jeu a été développé en équipe de quatre personnes, sur une durée de deux mois.

Outils employés :

  • Java et JavaFX
  • Git
  • Patron de conception MVC et Observers pour la communication entre l'utilisateur et le logiciel
  • Utilisation du TDD pour la création de code pertinent
  • Utilisation de SceneBuilder pour les interfaces du jeu
  • Utilisation des principes SOLID lors de la création du code

Compétences développées :

  • Externaisation de données et traitement sur celles-ci dans le logiciel(gestions de fichiers CSV avec java)
  • Utilisation de javaFX pour une interface responsive et design
  • Gestion des algorithmes de labyrinthe
  • Gestion des entrées utilisateurs au clavier
  • Création de tests adaptés et pertinents pour le projet

Golden Ace

SEPTEMBRE 2025

Mini projet Agile d'un jeu de casino

Voir le code

Description du projet:

Golden Ace est un mini-jeu de casino développé en Java dans le cadre d'un projet agile, jouable via le terminal. Le jeu fait commencer l'utilisateur avec une somme d'argent virtuelle par défaut, qu'il lui propose de parier sur différents mini-jeux qui lui apporteront gains ou pertes. Les fonctionnalités incluent la gestion des paris, le suivi du solde du joueur, et une interface utilisateur simple pour une expérience de jeu fluide.


Les différents jeux proposés sont:

  • Un Black Jack simple
  • Une roulette
  • Un démineur
  • Une machine à sous


Le projet a été réalisé en équipe de six personnes, choisies aléatoirement, sur une durée de quatre jours.

Outils employés :

  • Java
  • Git
  • Utilisation de la méthode Scrum et de la philosohie Agile durant toute la durée du projet
  • Utilisation du terminal pour l'interface utilisateur
  • Création de diagrammes UML pour la modélisation du projet
  • Répartition des tâches via un Kanban
  • Utilisation de LiveShare pour le travail collaboratif
  • Utilisation de tests unitaires pour la création de code pertinent

Compétences développées :

  • Externaisation de données et traitement sur celles-ci dans le jeu (gestion d'ASCII et fichier csv)
  • Apprentissage de la philosohie Agile
  • Travail en équipes aléatoires avec la méthode Scrum
  • Gestion des entrées utilisateur au clavier

PokéQuizAdventures

NOVEMBRE 2024 - JANVIER 2025

Création d'un jeu ludo-pédagogique

Voir le code

Description du projet:

PokéQuizAdventures est un jeu proposant d'incarner un pokémon et son dresseur qui partent à l'aventure. Pour combattre les ennemis ou soigner son pokémon, le joueur va devoir répondre à des questions de culture générale de différents niveaux de difficulté en fonction du nombre de points souhaités. La partie s'arrête lorsque le joueur a terminé tous les niveaux, que le pokémon n'a plus de vie, ou que le joueur décide de quitter le jeu.

Il s’agit d’un jeu implémenté en ijava, une version simplifiée du Java mais sans le concept Objet.


Le projet a été réalisé en binôme sur une durée de deux mois.

Outils employés :

  • iJava: pour un apprentissage du langage Java et de l'algorithmique
  • Git et LiveShare

Compétences développées :

  • Un apprentissage de l'algorithmique et de la logique de programmation
  • Travailler la mise en forme dansun terminal d'un programme java
  • Concevoir et conceptualiser des fonction génériques mais pertinentes
  • Apprendre à utiliser des plateformes collaboratives de travail tel que GitHub et LiveShare

Gestion d'une Base de Données des JO

MARS 2025 - MAI 2025

Création et exploitation d'une base de donnée des JO

Voir le code

Description du projet:

Le projet consistait à créer et exploiter une base de données relationnelle à partir des données des Jeux Olympiques, en utilisant le langage SQL. La base de données comprend plusieurs tables interconnectées, telles que Athlètes, Épreuves, Médailles, Pays, et Éditions des Jeux. Le tout a été modélisé à l'aide de diagrammes UML pour assurer une structure cohérente et efficace.


Le projet a été réalisé en binome sur une durée de trois mois.

Outils employés :

  • Access et Postgresql
  • Création de MCD et de MLD pour rendre la base optimisée
  • Language SQL

Compétences développées :

  • Création et gestion de bases de données relationnelles
  • Maîtrise du langage SQL pour le traitement et la manipulation de données
  • Conception de requêtes complexes pour extraire des informations pertinentes
  • Modélisation de données à l'aide de diagrammes UML
  • Analyse et interprétation de données réelles

La création de Site Web d'écomobilité

Voir le code

Description du projet:

Ubisoft LOCATION est un site Web vitrine. Ainsi, les utilisateurs allant sur le site ne peuvent pas réellement interagir avec e réelles fonctionnalités.


Il est basé sur la location de véhicules à mobilité vertes. Il se présente comme un site de la société Ubisoft, proposant à ses employés d’emprunter ou de louer leurs véhicules dans toute la France, aux autres collaborateurs de la marque.

Compétences développées :
