Mazel

SEPTEMBRE 2025 - NOVEMBRE 2025

Création d'un jeu de labyrinthe interactif
(Java et javaFX)

Description du projet:

Mazel est un jeu de labyrinthe à un joueur qui possède deux modes de jeu. Le premier est un mode progression accessible lorsqu'un utilisateur de "connecte" via un mot de passe reconue par le système, et ainsi accède à 8 niveaux différents, à difficultées progressives, qu'il doit compéter pour avancer dans les méandres du jardin de Mazel. Le second est un mode créatif de labyrinthes, personalisable par le joueur à l'aide de différentes options, comme entre autres : la taille du labyrinthe, le choix du type de labyrinthe, ou encore l'ajout d'd'un ennemi qui pourchasse le joueur.

Outils employés :

  • Java et JavaFX
  • Git
  • Patron de conception MVC et Observers pour la communication entre l'utilisateur et le logiciel
  • Utilisation du TDD pour la création de code pertinent
  • Utilisation de SceneBuilder pour les interfaces du jeu

Compétences développées :

  • Externaisation de données et traitement sur celles-ci dans le logiciel(gestions de fichiers CV avec java)
  • Utilisation de javaFX pour une interface responsive et design
  • Gestion des algorithmes de labyrinthe
  • Gestion des entrées utilisateurs au clavier
  • Création de tests adaptés et pertinents pour le projet

Golden Ace

SEPTEMBRE 2025

Mini projet Agile d'un jeu de casino (Java)

Description du projet:

Golden Ace est un mini-jeu de casino développé en Java dans le cadre d'un projet agile, jouable via le terminal. Le jeu fait commencer l'utiisateur avec une somme d'argent virtuelle de base, qu'il lui propose de parier dur différents jeux qui lui apporteront gains ou pertes. Les fonctionnalités incluent la gestion des paris, le suivi du solde du joueur, et une interface utilisateur simple pour une expérience de jeu fluide. Les différents jeux proposés sont:

  • Un Black Jack simple
  • Une roulette
  • Un démineur
  • Une machine à sous

Outils employés :

  • Java
  • Git
  • Utilisation de la méthode Scrum et de la philosohie agile durant toute la durée du projet (quatres jours)

Compétences développées :

  • Externaisation de données et traitement sur celles-ci dans le jeu (gestion d'ASCII et fichier csv)
  • Apprentissage de la méthodologie agile
  • Travail en équipe aléatoires avec la méthode Scrum
  • Gestion des entrées utilisateurs au clavier

PokéQuizAdventures

NOVEMBRE 2024 - JUIN 2025

Création d'un jeu ludo-pédagogique (Java/JavaFX)

Description du projet:

PokéQuizAdventures est un jeu proposant d'incarner un pokémon et son dresseur qui partent à l'aventure. Pour combattre les ennemis ou soigner son pokémon, le joueur va devoir répondre à des questions de culture générale de différents niveaux de difficulté en fonction du nombre de points souhaités. La partie s'arrête lorsque le joueur a terminé tous les niveaux, que le pokémon n'a plus de vie, ou que le joueur décide de quitter le jeu. Il s’agit d’un jeu implémenté en ijava, une version simplifiée du Java mais sans la partie développement orienté objet.

Outils employés :

  • iJava: pour un apprentissage du langage Java
  • GitHub et LiveShare

Compétences développées :

  • Un apprentissage de l'algorithmique et de la logique de programmation
  • Travailler la mise en forme console dans un programme java
  • concevoir et conceptualiser des fonction génériques mais pertinentes
  • Apprendre à utiliser des plateformes collaboratives de travail tel que GitHub et LiveShare

Gestion d'une Base de Données (SQL)

OCTOBRE 2024 - MAI 2025

Description du projet:

Le projet avait pour but d’apprendre à gérer une base de donnée. Il a ainsi permis de travailler le langage SQL, ainsi que ses différentes applications dans des cas concrets de données, la création de tables et la ventilation de celles-ci, et le traitement de données. Ces deux projets se sont réalisés en binômes.

Outils employés :

  • Access et Postgresql
  • SQL

Compétences développées :

  • Création et gestion de bases de données relationnelles
  • Maîtrise du langage SQL pour le traitement et la manipulation de données
  • Conception de requêtes complexes pour extraire des informations pertinentes

Ubisoft Location

SEPTEMBRE 2024 - NOVEMBRE 2024

La création de Site Web Vitrine (HTML/CSS)

Description du projet:

Ubisoft LOCATION est un site Web vitrine (les utilisateurs allant sur le site ne peuvent pas réellement interagir avec les fonctionnalités du site)
